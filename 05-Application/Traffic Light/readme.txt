Traffic Light Control

Traffic Light Control System Project with the specifications listed below:

1. STM32F103C8T6 Microcontroller with frequency 8Mhz.

2. The Seven segments will count 30 seconds while the red LED is on.

3. The Seven segments will count 20 seconds while the yellow LED is on.

4. The Seven segments will count 30 seconds while the green LED is on.

5. The project should be design and implemented based on the layered architecture model.
